# 서점 관리 프로그램
## 프로젝트 설명
JDBC프로그래밍을 활용하여 서점의 재고 및 회원의 정보, 마일리지 등을 관리하기 위해 제작되었다.

## 서비스 내용
- 서점 내 재고와 책 정보 관리
- 회원 관리 및 마일리지 관리

## 프로젝트 기간
2022.09 ~ 2022.12

## 개발 환경 및 언어
- Eclipse
- Java
- Oracle DB
- sql

## 주 활용 클래스
- Swing
- sql
